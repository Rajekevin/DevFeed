# 🚀 DevFeed — RSS Reader Moderne pour Développeurs

![Version](https://img.shields.io/badge/version-2.5-brightgreen)
![License](https://img.shields.io/badge/license-MIT-blue)
![Chrome](https://img.shields.io/badge/extension-Chrome-orange)

**DevFeed** est une extension de navigateur élégante qui transforme votre nouvel onglet en un hub d'actualités techniques. Suivez tous vos flux RSS préférés dans une interface moderne, rapide et entièrement personnalisable.

---

## ✨ Fonctionnalités

### 📰 Agrégateur RSS complet
- **Support multi-sources** - Ajoutez tous les flux RSS/Atom que vous voulez
- **Chargement parallèle** - Récupération rapide de tous vos flux simultanément
- **Cache local** - Persistance des articles dans localStorage
- **Sources par défaut** - PHP.net, React, Vue.js, Smashing Magazine, Dev.to, GitHub Blog, CSS Weekly, JavaScript Weekly

### 🎨 Design moderne
- **Thème clair/sombre** automatique avec persistance
- **Vue grille ou liste** selon vos préférences
- **Animations fluides** et transitions élégantes
- **Design responsive** - S'adapte à tous les écrans
- **Effet de grain subtil** pour un rendu premium

### 🏷️ Organisation avancée
- **Filtrage par catégorie** - PHP, JavaScript, Sécurité, DevOps, Design, Frontend, Backend
- **Filtrage par source** - Sélectionnez une source spécifique
- **Recherche de sources** - Trouvez rapidement une source dans la sidebar
- **Compteurs d'articles** - Visualisez le nombre d'articles par source/catégorie

### ⭐ Gestion personnelle
- **Marquer comme lu/non lu** - Suivi individuel de chaque article
- **Favoris** - Sauvegardez les articles importants
- **Statistiques** - Compteurs d'articles lus et sauvegardés
- **Streak** - Suivi de votre activité quotidienne

### 🔍 Recherche intégrée
- **Moteur DuckDuckGo** intégré directement dans l'interface
- **Respect de la vie privée** - Pas de tracking, pas de collecte de données

### ⚡ Performances
- **Pagination** - Navigation fluide dans les articles
- **Raccourcis clavier** - Flèches gauche/droite pour naviguer, Échap pour fermer les modales
- **Chargement asynchrone** - Interface non bloquante

## 📦 Installation

### Méthode 1: Extension Chrome (recommandée)

1. Clonez ou téléchargez ce repository
```bash
git clone https://github.com/rajekevin/devfeed.git
```


2. Ouvrez Chrome et allez sur chrome://extensions/

3. Activez le "Mode développeur" (en haut à droite)

4. Cliquez sur "Charger l'extension non empaquetée"

5. Sélectionnez le dossier contenant les fichiers de l'extension

6. L'extension est maintenant installée ! 🎉
